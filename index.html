// Selected Case Studies

Maizzle Responsive System

Maizzle Tailwind CSS Outlook 2019

A component-based email architecture built using Maizzle. This project solves the common "spaghetti code" issue in legacy email HTML by using utility classes that compile down to inline CSS. It specifically targets Outlook VML compatibility for background images and ensures bulletproof rendering in Dark Mode environments. This approach allows for rapid iteration of marketing campaigns without breaking layout consistency.

HTML5 Typography CTA Optimization

Designed with conversion in mind, this template utilizes a clean, single-column layout for mobile-first readability. Key features include a prominent "Quote Header" for social proof and ghost-button CTAs that degrade gracefully in clients that block images. The code structure is optimized for high deliverability, stripping out unnecessary bloat that triggers spam filters.

CompHealth Enterprise System

SOAP API Live Job Data Dynamic Content

This enterprise-grade system utilized complex SOAP scripting to interface directly with the Jobs API, fetching real-time vacancy data. The logic automatically filtered and populated job listings matching the specific specialty of the recipient (e.g., Locum Tenens vs. Permanent). The architecture supported over 20 distinct design variations—dynamically swapping headers, footers, and brand colors—allowing a single codebase to serve multiple sub-brands.
